# tg_session_creator

# Telegram Session Generator

Инструмент для удобного создания сессий Telegram с использованием библиотеки Telethon. Скрипт предназначен для упрощения настройки рабочих аккаунтов.

## Функционал

*   **Автоматизация API**: Возможность использования стандартных ключей API Telegram по умолчанию.
*   **Управление сессиями**: Сессии создаются в таком порядке: session1, session2 и т.д., что предотвращает перезапись существующих сессий.
*   **Методы авторизации**:
    *   QR-код: вход по сканированию QR-кода.
    *   Номер телефона: вход по коду подтверждения.
